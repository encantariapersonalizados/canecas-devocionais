# ☕ Canecas Devocionais

Este projeto é um site simples e leve criado para ser acessado através de **QR Codes impressos em canecas devocionais**.

Ao escanear o QR Code, o usuário é direcionado para uma página com um **devocional bíblico diário**, contendo:
- Versículo bíblico
- Reflexão
- Oração

O conteúdo muda automaticamente a cada dia, sem necessidade de login, banco de dados ou backend.

---

## 📖 Como funciona

- O site escolhe automaticamente **um devocional por dia**, com base na data atual.
- Cada dia do ano exibe um devocional diferente.
- O sistema evita repetição dentro do mesmo ano.
- No início de um novo ano, o ciclo recomeça.

Toda a lógica está implementada em JavaScript puro.
